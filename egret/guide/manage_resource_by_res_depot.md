# 使用Res Depot管理资源

`Res Depot`是`Egret`官方推出，管理资源文件的工具。它提供如下功能：

- 资源可视化管理
- 可预览图片
- 可播放音、视频
- 可检测到存在问题资源
- 提供一键清除不存在的资源
- 提供创建资源分组

借助它，可方便快捷管理资源文件，提高游戏开发效率。

## 2 添加资源到配置文件

- 拖拽文件到列表中

![add_file_01](../images/manage_resource_by_res_depot/add_files_01.png)

- 添加资源文件成功

![add_file_02](../images/manage_resource_by_res_depot/add_files_02.png)

- 添加资源文件失败，在列表中以及状态栏左下角又红色提醒。当保存配置文件时，会提醒无法保存，需要将错误信息处理后才能保存。

![add_file_fail](../images/manage_resource_by_res_depot/add_files_fail.png)

- 创建分组
  
![add_group_01](../images/manage_resource_by_res_depot/add_group_01.png)

- 从资源列表选择需要添加到分组文件，拖拽到分组的资源列表区

![add_group_02](../images/manage_resource_by_res_depot/add_group_02.png)

- 添加资源到分组成功，将会在分组的资源列表区显示
